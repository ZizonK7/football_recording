### v1.1.1 (2026-04-17)
**🛠️ 버그 수정 및 개선 (Bug Fixes & Improvements)**
* **북마크 시각적 식별성 강화:** 타임라인 및 리스트에서 북마크가 다른 일반 마커들과 확연히 구분되도록 표시 방식을 변경했습니다. 이제 중요한 지점을 훨씬 더 쉽게 식별할 수 있습니다.

* **시스템 안정성 향상:** 단축키 입력 반응 속도 및 처리 신뢰도를 높여 조작 편의성을 개선했습니다.
